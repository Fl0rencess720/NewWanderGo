## *请求方式都使用 GET*

  

## 早八数

##### 请求URL

- ` "/data/morning_eight" `
  

##### 返回示例

  

```
{
    "morning_eight":30
}

```

## 总课数

##### 请求URL

- ` "/data/classes_count" `
  

##### 返回示例

  

```
{
    "classes_count": 291
}

```

## 满课(九节)天数

##### 请求URL

- ` "/data/full_classes" `
  

##### 返回示例

  

```
{
    "full_classes": 0
}
```
